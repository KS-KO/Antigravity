# 프로젝트 요구사항 정의서 (PRD) - 해외직구 플랫폼 "Global Picks"

## 1. 프로젝트 개요
- **프로젝트 명**: Global Picks (글로벌 픽스)
- **목적**: 사용자가 해외 상품을 한국 쇼핑몰처럼 쉽고 안전하게 구매할 수 있는 환경 제공
- **핵심 가치**: 
    - **편리함**: 복잡한 배송대행지 입력 없이 바로 한국 주소로 배송
    - **투명성**: 실시간 환율 반영 및 관부가세 계산기 제공
    - **신뢰**: 검증된 해외 판매자 상품 노출 및 안전 결제 지원

---

## 2. 타겟 사용자
1. **해외 직구 초보자**: 해외 사이트 가입이나 배송대행지 이용이 어려운 20~40대
2. **가성비 중시 쇼핑객**: 국내보다 저렴한 해외 딜을 찾는 합리적 소비자
3. **트렌드 세터**: 국내에 출시되지 않은 최신 전자기기나 패션 아이템을 원하는 사용자

---

## 3. 핵심 기능 (MVP - Minimum Viable Product)

### 3.1 회원 및 인증
- **회원가입/로그인**: 이메일 기반 가입 및 카카오/네이버 소셜 로그인 지원
- **마이페이지**: 개인통관고유부호(PCCC) 관리, 주소지 관리, 주문 내역 확인

### 3.2 상품 탐색
- **홈 화면**: 추천 상품, 인기 카테고리, 타임딜(할인) 섹션
- **검색 및 필터**: 상품명 검색, 가격대/카테고리/브랜드 필터링
- **상품 상세**: 
    - 상품 이미지, 설명, 가격(현지 통화 + 원화 변환 가격)
    - 배송 예정일 및 배송비 안내
    - 사용자 리뷰 및 평점

### 3.3 주문 및 결제
- **장바구니**: 상품 담기, 수량 조절, 선택 삭제
- **결제 프로세스**: 
    - 한국 주소 입력 (우편번호 검색)
    - 개인통관고유부호 입력 필수 체크
    - 국내 PG(토스페이, 카카오페이 등) 연동 결제
- **관부가세 계산**: 결제 시 포함/미포함 여부 명시

### 3.4 사후 관리
- **주문 추적**: 해외 현지 배송 -> 항공 배송 -> 통관 -> 국내 배송 현황 실시간 업데이트
- **고객 지원**: 1:1 문의 채널 및 FAQ

---

## 4. 기술 스택 (추천)
신입 개발자가 이해하기 쉽고 유지보수가 용이한 최신 스택을 제안합니다.
- **Frontend**: Next.js (React 기반, SEO 최적화 및 빠른 로딩)
- **Styling**: Tailwind CSS (빠른 UI 개발 및 일관된 디자인)
- **State Management**: React Query (서버 데이터 캐싱 및 로딩 상태 관리)
- **Backend API**: Node.js (Express) 또는 Next.js API Routes
- **Database**: PostgreSQL (Prisma ORM 추천)
- **Payment API**: Portone (포트원) - 국내 결제 연동 표준

---

## 5. UI/UX 디자인 가이드라인
1. **모던 & 클린**: 흰색 배경에 포인트 컬러(예: 선명한 블루 #0066FF) 사용으로 신뢰감 전달
2. **반응형 디자인**: 모바일 사용자가 70% 이상임을 고려하여 Mobile-First로 설계
3. **마이크로 인터랙션**: 버튼 클릭 시 부드러운 애니메이션, 스켈레톤 로딩 등으로 체감 속도 향상
4. **접근성**: 폰트 크기와 대비를 충분히 확보하여 누구나 쉽게 읽을 수 있도록 구성

---

## 6. 개발 단계별 로드맵
- **1단계 (기획)**: 기능 명세 구체화 및 와이어프레임 작성
- **2단계 (디자인)**: 주요 화면 Mockup 제작
- **3단계 (기능 개발)**: 회원가입, 상품 목록, 상세 페이지 구현
- **4단계 (통합)**: 결제 연동 및 배송 추적 API 연결
- **5단계 (테스트/배포)**: QA(품질 보증) 및 Vercel/AWS를 통한 배포

---

## 7. 신입 개발자를 위한 팁
- **데이터 구조**: 상품 데이터에 `original_price` (현지 통화)와 `exchange_rate` (환율)를 어떻게 저장하고 보여줄지 고민해보세요.
- **통관 번호**: 한국 직구의 핵심은 '개인통관고유부호'입니다. 이를 정규표현식으로 검증하는 로직을 미리 준비하세요.
- **이미지 최적화**: 해외 사이트 이미지를 가져올 때 로딩 속도가 느려질 수 있으니 Next.js의 `Image` 컴포넌트를 적극 활용하세요.
